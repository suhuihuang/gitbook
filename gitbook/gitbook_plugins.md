# gitbook 插件

1. gitbook anchors 插件

   将Github样式标题锚添加到您的书中。



   ![hub](../images/hub.png)

   > 将插件添加到您的 book.json:

   ```javascript
   {
       "plugins" : [ "anchors" ]
   }
   ```

   * 用法：

     [gitbook 插件](# gitbook 插件)

     ```
     [gitbook 插件](# gitbook 插件)
     ```

     参考：<https://hltj.gitbooks.io/sandbox/content/index.html>
